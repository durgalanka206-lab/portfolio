# Lanka Durga Portfolio

Modern full-stack portfolio built with TanStack Start, React, TypeScript and Express.js.

## Project Structure

portfolio/
├── frontend/    # TanStack Start Frontend
└── backend/     # Express API + Nodemailer

## Tech Stack

Frontend:
- React 19
- TypeScript
- TanStack Start
- Vite

Backend:
- Node.js
- Express.js
- Nodemailer
